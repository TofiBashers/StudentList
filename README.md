StudentList
===========
Simple web application for create and modify the list of students. </br>
Supports token authorization, roles (user, admin).
</br>
</br>
Java version: 1.7 </br>

Backend technologies:
 - Spring Security
 - JAX-RS
 - Hibernate
 - H2 database (embedded)</br>

Frontend:
 - AngularJS
 - Twitter Bootstrap
